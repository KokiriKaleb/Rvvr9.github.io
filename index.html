HTML

Tag

Description

<DOCTYPE html>

declaration defines that this document is an HTML5 document

<html>

element is the root element of an HTML page

<head>

element contains meta information about the HTML page

<title>

element specifies a title for the HTML page

<body>

element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

<h1>

element defines a large heading

<p>

element defines a paragraph

Command Line

Tag

Description

pwd

displays the file path from the root directory to the current working directory

mkdir

used to make a new directory in the filesystem

ls

used to list the contents of a directory

cd

used to move throughout the filesystem of a computer

touch

creates a new file in the current working directory with the name provided
